## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

MODO DE USAR: 

Está disponível na API o Cadastro de Usuários, Funcionários e Patrimônios.

Exemplo (POST): 

http://localhost:3010/patrimonios

http://localhost:3010/usuarios

http://localhost:3010/funcionarios


Temos 2 métodos de pesquisa, por ID e a pesquisa geral que retorna todos cadastrados (Isso serve para Usuário, Funcionário e Patrimônio)

Exemplos (GET): 

http://localhost:3010/funcionarios/

http://localhost:3010/funcionarios/"aqui coloca o ID sem aspas"

http://localhost:3010/usuarios/

http://localhost:3010/usuarios/"aqui coloca o ID sem aspas"

http://localhost:3010/patrimonios/

http://localhost:3010/patrimonios/"aqui coloca o ID sem aspas"


Na API também já temos os módulos de remoção e alteração de Funcionário, Usuário e Patrimônio, porém, estes métodos não estão disponíveis no software por enquanto.
